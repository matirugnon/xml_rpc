# Documentación del Proyecto

Este directorio contiene la documentación técnica del proyecto XML-RPC sobre Sockets TCP/IP.

## Archivos

### [Grupo05.pdf](Grupo05.pdf)
Informe técnico completo del Grupo 05 que documenta:
- Diseño e implementación de la solución
- Decisiones tomadas durante el desarrollo
- Análisis de arquitectura y protocolo
- Pruebas y validación en red emulada
- Resultados y conclusiones

### [Obligatorio1-2025.pdf](Obligatorio1-2025.pdf)
Letra del obligatorio (enunciado del proyecto) que especifica:
- Requerimientos del proyecto
- Especificación del protocolo XML-RPC
- Criterios de evaluación
- Casos de prueba requeridos
- Formato de entrega
